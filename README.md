# Scoop Bucket for jig

This is the official Scoop bucket for [jig](https://github.com/toba/jig), Multi-tool CLI for issue tracking, citations, commits, and distribution.

## Installation

`powershell
scoop bucket add toba https://github.com/toba/scoop-jig
scoop install jig
`

## Usage

`powershell
jig version
`

## Updating

`powershell
scoop update jig
`

## Uninstalling

`powershell
scoop uninstall jig
scoop bucket rm toba
`

## Requirements

- Windows (amd64 or arm64)

## Issues
